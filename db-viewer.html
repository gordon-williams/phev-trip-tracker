BYD Energy Database Viewer

Extract and view trip data from your BYD head unit

Privacy: This page runs entirely in your browser. No data is uploaded to any server. The database file is read locally using JavaScript. You can verify this by disconnecting from the internet before selecting the file.

Select ec_database.db from the energydata folder
Also accepts renamed files (.jpg, .png)

How to find the file:
Head unit File Manager → navigate to energydata folder → find ec_database.db
If the file picker shows a different view, look in the internal storage or try searching for "ec_database"
